# 云桥Pro 行业模板与提示词库

版本: v1.0  
日期: 2026-06-28  
适配模型: `gpt-image-2`

## 1. 使用原则

云桥Pro 的提示词系统采用“行业模板 + 用户变量 + 输出约束 + 避免内容”的方式生成最终 prompt。这样既能保证用户操作简单，也能避免把不存在的 API 参数误做成模型参数。

最终 prompt 推荐结构:

```text
任务目标 + 主体描述 + 场景/风格 + 构图 + 光线 + 材质/细节 + 用途约束 + 需要保留的内容 + 需要避免的内容
```

通用变量:

- `{subject}`: 主体
- `{product_name}`: 商品名
- `{brand}`: 品牌名
- `{selling_points}`: 卖点
- `{scene}`: 场景
- `{style}`: 风格
- `{color_palette}`: 色彩
- `{audience}`: 目标人群
- `{text}`: 需要出现在图中的文字
- `{platform}`: 平台
- `{season}`: 节日/季节
- `{project}`: 项目名

通用避免内容:

```text
避免水印、二维码、乱码文字、多余 logo、侵权品牌标识、畸形结构、严重透视错误、低清晰度、脏污背景、过度锐化、过曝、欠曝、重复主体、边缘残缺、明显 AI 伪影。
```

人物类额外避免:

```text
避免面部崩坏、五官错位、多余手指、手部畸形、肢体比例异常、皮肤塑料感、眼神空洞、重复人物、身份特征不一致。
```

商品类额外避免:

```text
避免改变商品外形、改变颜色、改变包装文字、凭空增加配件、产品变形、材质错误、商标错乱、反光过曝、边缘融化。
```

文字建议:

- 精准标题、价格、活动字样优先由软件图层叠加。
- 若必须让模型生成文字，prompt 中使用“只渲染以下文字: ...，不要生成其他文字”。

## 2. 电商零售

### 2.1 白底商品主图

推荐尺寸: `1024x1024`  
推荐质量: `medium`，终稿 `high`  
推荐格式: `png`

```text
为电商平台生成一张商用商品白底主图。主体是 {product_name}，产品特征为 {selling_points}。保持商品外形、颜色、包装和材质真实准确，纯白干净背景，柔光棚拍，居中构图，阴影自然，边缘清晰，细节锐利，适合淘宝、京东、拼多多上架。避免改变商品结构、虚假 logo、水印、乱码文字、杂乱背景、过曝反光、产品边缘变形。
```

### 2.2 商品场景种草图

推荐尺寸: `1024x1536`

```text
生成一张电商种草场景图。主体是 {product_name}，突出 {selling_points}。场景为 {scene}，整体风格 {style}，画面有真实生活氛围，商品位于视觉中心，背景干净但不空洞，光线柔和自然，色彩高级，适合详情页和小红书种草。保持商品外观准确，不要改变品牌包装，不要添加无关配件。
```

### 17.1 高保真轻微修改

```text
基于输入图片进行轻微修改。请尽量保持主体、构图、颜色、材质、人物身份或商品结构不变，只调整以下内容: {subject}。修改后画面自然一致，看不出编辑痕迹。避免改变主体比例、脸部身份、商品外形、背景逻辑。
```

### 17.2 商品换背景

```text
基于输入图片编辑。保持商品主体的外形、颜色、包装、材质和角度完全一致，将背景替换为 {scene}。新背景要与商品光线、阴影、透视自然融合，适合商用电商图。不要改变商品本身，不要添加额外 logo 或文字。
```

### 17.3 人物换场景

```text
基于输入人物图片编辑。保持人物身份、五官、发型、服装和姿态尽量一致，将场景更换为 {scene}，整体风格 {style}。光线和色调要自然融合，看起来像同一次拍摄。避免改变人物身份、脸型、年龄、服装结构。
```

### 17.4 局部重绘

```text
只重绘遮罩区域，未遮罩区域必须保持不变。遮罩区域需要修改为 {subject}，风格、光线、透视和材质与原图一致，边缘自然融合。不要影响遮罩外的主体、背景、文字或构图。
```

### 17.5 扩图

```text
基于输入图片进行画面扩展。保持原图主体、光线、视角、色彩和风格一致，将画面向外自然延展，补全合理背景和环境细节。不要改变原图主体，不要让扩展区域出现重复、断裂或风格不一致。
```

# 云桥Pro UI 设计规范

版本: v1.0  
日期: 2026-06-28

## 1. 设计目标

云桥Pro 的界面应像专业生产工具，而不是营销落地页。视觉要沉稳、耐看、信息密度高，适合用户连续工作数小时。重点是参数清晰、图片管理高效、批量任务可控、右侧操作始终触手可及。

## 2. 全局布局

固定三分屏:

- 顶部通栏: 64px
- 左侧导航: 240px
- 中间创作区: 自适应
- 右侧预览与操作: 360px
- 底部状态栏: 28px

最小窗口:

- Windows: 1280 x 800
- 推荐: 1440 x 900 及以上

响应规则:

- 宽度小于 1366 时，右侧面板可收起为图标栏。
- 宽度小于 1200 时，左侧导航变为仅图标模式。
- 所有参数面板支持纵向滚动，底部生成按钮固定。

## 4. 字体

- Windows: Microsoft YaHei UI, Microsoft YaHei
- macOS: PingFang SC
- 数字: Inter 或系统默认

字号:

- 页面标题: 20px/28px, 600
- 模块标题: 16px/24px, 600
- 正文: 14px/22px, 400
- 表单标签: 13px/20px, 500
- 辅助说明: 12px/18px, 400
- 状态栏: 12px/16px, 400

不要使用随窗口宽度变化的字体尺寸。

## 5. 圆角和间距

- 按钮: 8px
- 输入框: 8px
- 图片缩略图: 8px
- 卡片和列表项: 12px
- 弹窗: 16px
- 工具栏按钮: 8px

间距:

- 页面外边距: 20px
- 面板内边距: 16px
- 表单项间距: 14px
- 按钮间距: 8px
- 列表项间距: 10px

## 6. 顶部通栏

左侧:

- Logo
- 产品名: 云桥Pro AI绘图
- 版本标签: GPT-Image-2 商用专业版

中间:

- 搜索框，占位: 搜索模板/关键词/历史作品

右侧:

- 模型显示: `gpt-image-2`
- 质量档位选择
- 云同步状态
- API 状态
- 会员/额度
- 设置
- 最小化
- 最大化
- 关闭

交互:

- API 异常时顶部显示红色状态点。
- 云同步进行中显示进度环。
- 搜索聚焦后展示最近模板、最近项目、最近作品。

## 7. 左侧导航

导航项:

1. 工作台
2. 文生图创作
3. 图生图与局部重绘
4. 批量生产工坊
5. 行业模板库
6. 作品素材库
7. AI 修图工具箱
8. API 与云端存储设置

每项包括:

- 图标
- 中文名称
- 未完成任务数量角标

选中状态:

- 左侧 3px 品牌渐变条
- 背景 `rgba(59, 130, 246, 0.12)`
- 文字白色或深色主题主文字

## 8. 中间创作区

文生图页面结构:

- 顶部: 行业和模板选择
- 主输入区: prompt 编辑器
- 参数区: 尺寸、质量、格式
- 高级区: 输出约束、避免内容、云上传、项目归档
- 底部固定: 生成按钮、预估任务数、重置

图生图页面结构:

- 左侧画布
- 右侧参数
- 底部缩略图输入带
- 顶部工具栏: 选择、画笔、橡皮、缩放、撤销、重做、适配窗口

批量页面结构:

- 顶部导入区域
- 中间任务表格
- 右侧批量参数
- 底部队列控制

## 9. 右侧预览面板

固定宽度 360px。

区域:

- 当前大图预览
- 缩略图列表
- 操作按钮
- 云端信息
- 参数快照
- 迭代历史

按钮:

- 载入图生图继续创作
- 复制云端 URL
- 复制提示词
- 下载
- 上传云端
- 加入收藏
- 删除

图片 hover:

- 缩放 1.03
- 阴影增强
- 显示快捷按钮

## 10. 表单控件

尺寸控件:

- 常用比例按钮: 1:1、3:4、4:3、9:16、16:9、2K
- 自定义输入: 宽、高
- 校验: 宽高必须是 16 的倍数，比例在 1:3 到 3:1 之间

质量控件:

- 自动: `auto`
- 草稿: `low`
- 标准: `medium`
- 精修: `high`

数量控件:

- 当前暂不开放多图数量，单次请求默认 1 张
- 批量生产通过多条任务逐张生成

格式控件:

- PNG: 精修、二次编辑推荐
- JPEG: 上架和预览推荐
- WebP: 小体积网页推荐

压缩控件:

- 仅 JPEG/WebP 显示
- 范围 0 到 100

## 11. 动效

- 按钮 hover: 亮度提升，品牌按钮轻微外发光
- 按钮 active: scale(0.98)
- 弹窗: 160ms scale + fade
- 图片 hover: 180ms scale
- 进度条: 品牌渐变流动
- 队列抽屉: 220ms slide

动画应克制，不能影响长时间工作。

## 12. 弹窗规范

通用弹窗:

- 标题
- 简短说明
- 主内容
- 取消按钮
- 主操作按钮

高风险弹窗:

- 删除作品
- 清空缓存
- 移除云存储配置
- 仅云端保存模式

删除确认文案:

```text
确认删除该作品？
删除后本地文件将移入回收站，作品记录会从素材库移除。云端文件不会自动删除。
```

## 13. 空状态

工作台无作品:

```text
还没有作品。选择一个行业模板，开始第一张商用图。
```

素材库无搜索结果:

```text
没有找到匹配作品。换个关键词或清除筛选条件。
```

批量工坊未导入:

```text
导入 Excel、CSV，或直接粘贴多行关键词开始批量生产。
```

云端未配置:

```text
配置云端存储后，生成图片会自动上传并返回公网 URL。
```

## 14. 状态提示

生成中:

```text
正在生成第 {current}/{total} 张，预计还需 {seconds} 秒
```

上传成功:

```text
已上传到 {storageName}，云端 URL 可复制
```

尺寸错误:

```text
宽高需要是 16 的倍数，且长短边比例不能超过 3:1
```

请求失败:

```text
生成失败。请检查 API Key、网络代理或稍后重试。
```

## 15. 可访问性

- 所有图标按钮必须有 tooltip。
- 所有关键按钮支持键盘焦点。
- 深色主题文字对比度不低于 WCAG AA。
- 图片缩略图选中不能只依赖颜色，还要有边框和勾选标识。

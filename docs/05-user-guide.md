# 云桥Pro v0.1.10 使用说明

## 1. 客户端下载

GitHub Release 页面：

```text
https://github.com/jxb412/YunQiao-Image-Studio/releases
```

v0.1.10 提供三个客户端：

- Windows x64：`YunQiao-Image-Studio-0.1.10-win-x64-portable.exe`
- macOS Intel x64：`YunQiao-Image-Studio-0.1.10-mac-x64.dmg`
- macOS Apple Silicon arm64：`YunQiao-Image-Studio-0.1.10-mac-arm64.dmg`

Windows 版本是便携版，下载后双击 exe 即可运行，不需要安装 Node.js、npm 或 Electron。

### Windows 首次运行提示

当前开源版本未做 Windows 代码签名，首次运行便携版时可能出现 Microsoft Defender SmartScreen 提示：

```text
Windows 已保护你的电脑
发布者：发布者未知
```

如果确认客户端来自本项目 GitHub Release，可以按下面方式运行：

1. 点击 `更多信息`。
2. 点击 `仍要运行`。
3. 程序会正常启动。

这是 Windows 对未签名应用的安全提示，不代表程序损坏。正式商业分发建议使用代码签名证书重新签名发布。

## 2. API 设置

应用固定使用：

```text
https://api.0029.org
```

用户需要先到 `0029.org` 购买套餐并生成秘钥，然后在 `API与云端存储设置` 页面保存 API Key。

设置页支持：

- 保存 API Key 到本机安全存储。
- 请求超时时间设置，默认 300 秒，最高 600 秒。
- 测试 API 连通性，显示 HTTP 状态、耗时、接口返回摘要和错误信息。

API Base URL 不提供自定义修改入口。

## 3. 文生图

进入 `文生图创作`：

1. 填写项目名称。
2. 选择行业模板。
3. 填写业务变量，例如产品名、卖点、场景、风格等。
4. 检查尺寸、质量、输出格式和背景参数。
5. 点击 `生成图片`。

当前默认每次生成 1 张图片，不传多图数量参数，避免中转接口把数量转为 `tools[0].n` 后被官方接口拒绝。

生成完成后会显示：

- 状态
- 来源
- 耗时
- 返回类型
- 图片类型
- 图片尺寸
- 文件大小
- 本地路径
- 云端上传状态

## 4. 图生图与连续编辑

进入 `图生图与局部重绘`：

1. 点击底图区域选择本地图片，或从作品库点击 `继续编辑`。
2. 修改提示词，说明要保留什么、修改什么。
3. 点击 `开始图生图`。
4. 生成结果会自动保存，并可继续作为下一轮底图。

结果区提供：

- 继续图生图
- 载入 AI 修图
- 保存图片
- 上传云端
- 复制图片
- 打开本地文件位置
- 大图预览
- 对比视图
- 迭代链

## 5. AI 修图工具箱

AI 修图工具箱包含：

- AI 扩图
- 去杂物
- 换背景
- 统一色调
- 商品光影
- 人像精修
- 局部重绘
- 去文字水印
- 老照片修复
- 封面增强

局部遮罩使用流程：

1. 选择要修的原图。
2. 点击 `绘制局部遮罩`。
3. 使用画笔涂抹需要修改的区域。
4. 可撤销、重做、清空。
5. 点击 `使用遮罩`。
6. 下一次修图请求会把遮罩一起提交给图片编辑接口。

## 6. 批量生产工坊

批量 Excel 列顺序：

```text
项目、行业、模板、数量、提示词、尺寸
```

批量功能：

- 导入 Excel。
- 下载 Excel 模板。
- 创建样例任务。
- 每条任务单独绑定底图。
- 从文件夹批量绑定底图，按任务顺序循环分配。
- 有底图走图生图，没有底图走文生图。
- 失败自动重试，重试次数可设置。
- 队列支持开始、暂停、继续、停止。
- 支持重试失败任务、清理完成任务、移除任务。
- 支持导出任务日志。

## 7. 行业模板库

模板库支持：

- 按行业筛选。
- 关键词搜索。
- 推荐模板筛选。
- 查看模板详情。
- 复制提示词。
- 使用模板进入创作页。
- 新增或编辑模板。
- 导入/导出 JSON 模板包。

导入模板会作为自定义模板保存到本机。

## 8. 作品素材库

作品素材库支持：

- 搜索作品、项目、提示词、云端 URL。
- 按项目筛选。
- 按云端状态筛选。
- 大图预览。
- 复制提示词。
- 上传云端。
- 继续编辑。
- 载入修图。
- 打开本地文件位置。
- 导出 URL 清单。
- 打包素材压缩包。

## 9. 云端存储

支持的存储：

- 阿里云 OSS
- 腾讯云 COS
- 华为 OBS
- 七牛云
- MinIO
- FTP
- SFTP

设置页支持网络连通性测试和真实上传测试。上传成功后会返回对象路径、公网 URL、耗时等信息。

## 10. macOS 首次打开

当前开源构建未做 Apple Developer ID 签名和公证，macOS 可能提示无法验证开发者。

推荐方式：

1. 打开 DMG，把应用拖到 `Applications`。
2. 在 `Applications` 中右键点击 `云桥Pro.app`。
3. 选择 `打开`。
4. 弹窗提示无法验证时，再次点击 `打开`。

如果仍提示无法打开，可在终端执行：

```bash
xattr -cr /Applications/云桥Pro.app
```

仅信任来源软件才可以这样处理。

## 11. 检查更新

顶部工具栏提供检查更新入口。点击后会检查 GitHub 最新 Release，并打开下载页面。
